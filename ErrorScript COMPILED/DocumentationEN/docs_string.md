# string
The **string** datatype is a type for objects that can hold letters, numbers, and symbols.

## Operators

### [`string`](docs_string.md) + [`string`](docs_string.md)
Adds two **string**s together.

Example:
````js
let hello = 'Hello, '
let world = 'World!'

print( hello + world ) // Prints 'Hello, World!'
````
